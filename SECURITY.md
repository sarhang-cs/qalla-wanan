# Security

To report a security issue privately, email `sarhang.salah9@gmail.com` with reproduction steps and affected URL/version. Do not publish credentials, private user data, Supabase secret/service-role keys, MySQL passwords, or access tokens in GitHub issues.

Client-side configuration may contain only public browser keys such as a MapTiler key restricted to approved domains and a Supabase Publishable key protected by Row Level Security. Elevated Supabase Secret/service-role keys and MySQL credentials belong only in server-side configuration.
