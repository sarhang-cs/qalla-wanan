# Qalla Wanan — R15

لە R15 ـدا فۆنتی ڕووکاری گشتی ئەپەکە هەروا `Vazirmatn / Noto Kufi Arabic` ماوەتەوە. تەنها ناوی شوێنەکانی native ـی MapLibre بە فۆنتی `UniQAIDAR_Money Heist 002` نیشان دەدرێن.

## گۆڕانکارییەکان

- قەبارەی ناوی شار، گوند، ڕێگا، دوکان و POI بە شێوەی بەرچاو گەورە و خوێنراو کرا
- پچرانی RTL ـی کوردی و عەرەبی بە NFC normalization و RTL plugin پارێزرا
- بۆکسە دارکەکان native و فشردەن و بە قەبارەی دەق دەگونجێن
- شلوغی بە zoom و collision priority ڕێکخرا؛ هەموو 69,000 تۆمار لە داتا و Search ـدا ماون
- هیچ DOM/Canvas overlay ـێک نییە؛ ناوەکان native WebGL symbol ـن
- coordinate، mask، Satellite، GPS و route پارێزراون

## فۆنت

فایلی `UniQAIDAR-Money-Heist-002.ttf` لە Downloads بهێڵەوە. سکریپتی Termux خۆکارانە دەیدۆزێتەوە، دەیخاتە `public/fonts/`، build دەکات و لەگەڵ پڕۆژەکە push دەکات. هیچ هەنگاوی دەستی بۆ فۆنت پێویست نییە.
